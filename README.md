## Usage
Build the work with Android Studio and export as APK file, install it on your phone.
Or you can find the prebuild `.apk` file from [here](./res/app-debug-androidTest.apk)

### Repo structure
```bash
├── app
│   ├── build
│   │   ├── generated
│   │   ├── intermediates
│   │   ├── outputs
│   │   └── tmp
│   ├── libs
│   └── src
│       ├── androidTest

│       └── test

└── res
```
